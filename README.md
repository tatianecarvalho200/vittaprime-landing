# VittaPrime Landing Page

Projeto em React + Vite para landing page da clínica VittaPrime.

## Rodar localmente

```bash
npm install
npm run dev
```

## Onde trocar informações

- WhatsApp: `src/data/content.js` na constante `WHATSAPP_NUMBER`
- Textos: `src/data/content.js`
- Instagram: `src/data/content.js`
- Endereço e mapa: `src/data/content.js`
- Imagens: pasta `src/assets`

## Importante

As imagens da pasta `src/assets` são placeholders. Substitua pelas fotos reais da clínica mantendo exatamente os mesmos nomes.
